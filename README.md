
# 湖南全域(Hunan Map Guide)

这是一个基于 Web 的交互式地图应用，旨在展示湖南省（特别是娄底地区）的现代景点分布与历史行政区划变迁。项目使用原生 JavaScript 和 Leaflet.js 构建，轻量高效，无需复杂的后端依赖。

## 🚀 功能特性

### 1. 双模式视图

* **🚀 现代景点 (Modern Tour)**
* 展示高校、风景名胜、文化遗址等 POI 数据。
* 提供详细的景点图文介绍及导航链接。


* **📜 历史疆域 (Historical Territory)**
* 通过时间轴展示从秦汉时期到现代的行政区划演变。
* 重点展示梅山地区、湘南县到现代娄底市的建制变迁。



### 2. 交互式地图体验

* **多层级下钻**：支持从“湖南全省”概览点击进入“市州”详情，再深入查看“区县”板块。
* **双底图切换**：集成高德电子地图与卫星影像，满足不同场景需求。
* **平滑动画**：行政区切换与视角定位均包含平滑过渡动画。

### 3. 数据检索与过滤

* **分类筛选**：支持按“全部”、“高校”、“特定区县”快速筛选景点。
* **关键词搜索**：支持实时检索景点名称或描述内容。
* **智能配色**：根据行政区划自动生成区分度高的色彩方案。

## 📂 项目结构

```text
hunan-map/
├── index.html              # 应用入口文件 (UI 结构)
├── style.css               # 样式表 (布局、动画、响应式适配)
├── editor.html             # (可选) 辅助编辑工具
├── images/                 # 景点图片资源文件夹
├── city/                   # GeoJSON 地理数据文件夹

│   ├── app.js              # 核心逻辑 (地图初始化、交互、渲染)
│   ├── data.js             # 景点数据 (spots 数组) & 历史时期配置
│   ├── geo-data.js         # 地理数据配置索引
│   └── history.js          # 历史地图渲染模块
└── README.md               # 项目说明文档

```

## 🛠️ 技术栈

| 类型 | 技术/库 | 说明 |
| --- | --- | --- |
| **核心** | HTML5 / CSS3 | 响应式布局，适配移动端与桌面端 |
| **逻辑** | JavaScript (ES6+) | 原生 JS 实现，无构建步骤 |

## 📦 安装与运行

由于项目使用 ES Modules 和 Fetch API 加载本地 JSON 资源，直接双击 `index.html` 可能会因浏览器的 **CORS 策略**（跨域限制）导致数据加载失败。

**推荐运行方式：**

1. **环境准备**：确保已安装 [Node.js](https://nodejs.org/)。
2. **安装服务**：在项目根目录下安装简易 HTTP 服务器（例如 `http-server`）：
```bash
npm install -g http-server

```


3. **启动服务**：
```bash
http-server .

```


4. **访问应用**：在浏览器中打开命令行提示的地址（通常为 `http://localhost:8080`）。

> **提示**：如果您使用 VS Code，也可以安装 **Live Server** 插件，右键 `index.html` 选择 "Open with Live Server" 即可一键运行。

## 🗺️ 数据来源

* **地图底图**: 高德地图 (AutoNavi) Tile Layer
* **边界数据**: 阿里云 DataV GeoAtlas
* **景点数据**: 手动采集与整理 (详见 `js/data.js`)
